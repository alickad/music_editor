Simple Music Notation Editor

This is a simple notes editor working on javascript. If you want to see the code (at your own risk, I am not responsible for the trauma caused), or you would like to see a bit more technical documentation, it is on github: github.com/alickad/music_editor. Here is a guide on how to use the tool. Enjoy :)

Adding notes

There are two ways how you can add notes:

1. Select from dropdown menus

  • select the desired note, octave, accidental and duration
  • click "Add Note" button
  • if you select the note to be a rest, octave and accidentals info will be ignored
  • you can only add one note at a time. If you want to create chords, use the option to add notes from keyboard
  • to erase the last added note, hit Backspace

    • 2. Add from keyboard

  • Type the desired notes into the "Type note" window, then hit Enter.
  • To delete the last added note, click outside the typing window and hit Backspace.

    • Format of notes in keyboard mode

  • letter representing a note (upper or lower case)
  • valid accidentals if needed (#, ##, b, bb, n for sharp, double sharp, flat, double flat and natural)
  • backslash followed by octave. If no octave is written, 4 is automatically assigned.
  • space and number representing duration (1 is a whole, 2 is a half-note, 4 is a quarter-note and so on). If the number is missing, 4 is automatically assigned.

    • Chords

    To form a chord, write the desired notes devided by Space, each of them as described above. The duration has to be written last and is the same for the whole chord.

    Rests

    Simply write "rest" and the number representing duration.

    Other functions

  • Delete all the notes by clicking the "Clear" button
  • For export, select your preffered format in the dropdown selection next to "Download" option
  • Listen to the music by clicking "Play"
