# music_editor
A simple music notation web editor. Some version (not neccessarily up to date) can be found on https://www.ms.mff.cuni.cz/~domanyoa/music_editor/index.html

## Functions
- very basic note scripting
- download
- playback

## Used libraries
- VexFlow (notes rendering)
- TuneJS (playback)
